# SEO editorial roadmap (12–18 months)

## North-star structure
- **Q1:** Pillar + pain + template
- **Q2:** Pain + VC pages + case examples
- **Q3:** Case studies + transition pages
- **Q4:** Whitepaper + benchmarking report

## Quarterly plan

### Q1 (Months 1–3): Authority foundation
- Publish/expand authority pillars for capital discipline and board posture.
- Pair each pillar with one pain page and one template/tool artifact.
- Priority outcomes:
  - Canonical topical ownership for governance language.
  - Baseline CTA conversion instrumentation on every long-form page.

### Q2 (Months 4–6): VC governance expansion
- Publish VC-focused authority page: `/resources/how-vc-firms-can-enforce-capital-discipline-across-portfolios`.
- Publish case-example support pages for portfolio governance and posture standardization.
- Priority outcomes:
  - Portfolio governance keyword footprint growth.
  - Comparable board-forwardable language across VC/operator pages.

### Q3 (Months 7–9): Case-study depth + transition intent
- Expand from case examples into structured case studies.
- Add transition pages (e.g., “from freeze posture to selective offense”) that connect pain content to implementation content.
- Priority outcomes:
  - Higher assisted conversion rate from internal links.
  - Better persona-to-content match by intent tag.

### Q4 (Months 10–12): Signature assets
- Publish annual whitepaper on capital posture governance quality.
- Publish benchmarking report (portfolio posture maturity by segment).
- Priority outcomes:
  - Backlink quality and citation lift.
  - Executive/board lead quality improvements.

## Months 13–18 extension
- Refresh top quartile pages monthly.
- Reissue whitepaper/report with new benchmark cohorts.
- Build one comparative hub page for every winning cluster path (Authority/Pain/Tool/VC).
