# Developer docs

Public integration docs for Whether without requiring Mintlify.

## Start here
- `quickstart.md` — get your first successful API call in under five minutes.
- `api-reference.md` — endpoint catalog, request examples, and response expectations.
- `integrations.md` — agent workflow, alerts/delivery flows, and production monitoring.

## Intended audience
- Platform engineers integrating Whether into internal tools.
- AI/agent developers that need machine-readable summaries + provenance.
- Ops teams wiring alerts into Slack/email/webhook workflows.

## Scope and source of truth
These pages summarize external integration workflows and point back to implementation source files where needed.
For contributor workflow and implementation policy, use `../development-playbook.md` and `../agents/README.md`.
